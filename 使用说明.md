# 和金顺数据平台 - 使用说明

## 系统要求

- Windows 10 或更高版本
- Chrome 浏览器（用于数据采集）
- Python 3.8+（如果使用源码运行）

## 使用方法

### 方式1：直接运行打包文件

1. 双击 `dist\和金顺数据仪表盘.exe`
2. 等待浏览器自动打开
3. 在网页中进行操作

### 方式2：使用源码运行

```bash
python main.py
```

然后在浏览器中打开 http://127.0.0.1:5001

## 功能说明

### 数据接入（核心功能）

1. **输入数据源地址**
   - 支持京东搜索页面URL
   - 示例：`https://search.jd.com/Search?keyword=漆器`

2. **设置采集数量**
   - 建议5-20条
   - 不要设置太大，避免被反爬虫拦截

3. **点击"开始采集"**
   - Chrome浏览器会自动打开
   - 查看瀑布流日志了解进度
   - 等待采集完成

4. **查看数据**
   - 采集完成后点击"查看数据看板"
   - 查看可视化图表

### 数据看板

- 查看所有已采集的商品数据
- 支持价格分布、平台占比等可视化分析

### 自定义分析

- 使用代码生成器创建自定义图表
- 粘贴JSON配置进行渲染

## 注意事项

1. **首次使用**
   - 可能需要允许防火墙访问
   - Chrome浏览器需要正常安装

2. **数据采集**
   - 不要频繁大量采集
   - 如遇到反爬虫，等待几分钟后重试
   - 建议每次采集不超过20条

3. **浏览器窗口**
   - 采集过程中Chrome窗口会自动打开
   - 不要关闭浏览器窗口
   - 不要手动操作浏览器

## 故障排除

### 问题1：端口被占用

```
解决：关闭其他占用5001端口的程序
或修改main.py中的端口号
```

### 问题2：Chrome浏览器无法打开

```
解决：确保Chrome已正确安装
或检查杀毒软件是否拦截
```

### 问题3：数据采集失败

```
解决：
1. 检查网络连接
2. 确认URL格式正确
3. 等待几分钟后重试
4. 减少采集数量
5. 首次使用可能需要手动登录京东
   - Chrome浏览器会自动打开
   - 在浏览器中登录京东账号
   - 登录成功后系统会自动继续
```

### 问题4：反爬虫拦截

```
解决：
1. 等待10-15分钟后重试
2. 更换网络环境（如使用手机热点）
3. 手动登录京东后再采集
4. 减少采集数量（建议每次不超过10条）
```

## 技术支持

- 项目位置：`C:\Users\Administrator\Desktop\和金顺新计划`
- 日志位置：控制台输出
- 数据存储：`data/cleaned/products.xlsx`
