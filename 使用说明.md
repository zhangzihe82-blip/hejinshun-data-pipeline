# 和金顺数据平台 - 使用说明

## 项目简介

和金顺数据平台是一站式电商数据采集、生成与可视化平台，包含四大核心功能：

- **数据看板** -- 电商数据可视化监控，支持价格分布、平台占比、用户画像等图表
- **数据接入** -- 关键词搜索或URL输入，从京东、什么值得买采集商品数据
- **数据工厂** -- 结论驱动数据生成，支持结构化参数和自然语言两种输入方式
- **代码生成器** -- 上传Excel文件自动生成ECharts可视化配置代码

---

## 功能列表

### 1. 数据看板 (端口 5001)

| 功能 | 说明 |
|------|------|
| 商品数据列表 | 展示所有已采集/生成的商品，按时间、价格、名称排序 |
| 平台占比分析 | 京东、京东自营、什么值得买的商品数量分布饼图 |
| 价格分布图 | 商品价格区间分布与统计 |
| 用户画像统计 | 买家年龄分布、性别比例、地区分布 |
| 统计概览 | 商品总数、平均价格、最高/最低价格等汇总信息 |

### 2. 数据接入 (端口 5001 内)

| 功能 | 说明 |
|------|------|
| 关键词搜索 | 输入关键词选择平台（京东/什么值得买），自动构建搜索URL并采集 |
| URL采集 | 直接粘贴京东或什么值得买的搜索页URL |
| 采集数量 | 每次可采集1-200条数据 |
| 实时日志 | 采集过程中实时显示进度和状态信息 |
| 停止采集 | 支持随时中断正在进行的采集任务 |

**支持的平台：**

| 平台 | 关键词 | 示例URL |
|------|--------|---------|
| 京东 | `jd` | `https://search.jd.com/Search?keyword=漆器` |
| 什么值得买 | `smzdm` | `https://search.smzdm.com/?c=home&s=漆器` |

京东爬虫还支持以下页面类型的自动识别与采集：
- 搜索页 (search.jd.com)
- 列表页 (list.jd.com)
- 商品详情页 (item.jd.com)
- 店铺页 (mall.jd.com)
- 短链接 (re.jd.com)
- 移动端页面 (m.jd.com)

### 3. 数据工厂 (端口 5007)

| 功能 | 说明 |
|------|------|
| 结构化输入 | 设置品类、价格区间、年龄范围、性别比例等参数 |
| 自然语言输入 | 输入结论语句，系统自动解析并生成支持该结论的数据 |
| 输出格式 | 支持 Excel (.xlsx)、JSON、CSV 三种格式 |
| 一键导出 | 将生成的数据直接合并到平台主数据文件 |
| 下载文件 | 下载生成的数据文件到本地 |

**自然语言输入示例：**

```
手机品类价格呈上涨趋势
京东自营评分明显高于普通京东
美妆品类评论数最高
高价位商品评分更稳定
```

**结构化输入参数：**

| 参数 | 说明 | 示例 |
|------|------|------|
| 品类 | 商品分类 | 手机、电脑、家电、服装、食品、图书、美妆、家居 |
| 价格区间 | 最低价-最高价 | 100-5000 |
| 年龄范围 | 最小年龄-最大年龄 | 18-35 |
| 女性占比 | 性别比例 | 65% |
| 生成数量 | 1-10000条 | 500 |
| 随机种子 | 可复现的随机数种子 | 42 |

### 4. 代码生成器 (端口 5003)

| 功能 | 说明 |
|------|------|
| Excel上传 | 上传一个或多个 .xlsx/.xls 文件 |
| 自动识别图表类型 | 根据文件名关键词自动匹配图表类型 |
| ECharts配置生成 | 输出完整的ECharts可视化配置代码 |

**支持的图表类型与文件名关键词：**

| 图表类型 | 文件名包含关键词 | 示例文件名 |
|----------|-----------------|-----------|
| 饼图 | 饼图 | 平台占比饼图.xlsx |
| 环形图 | 环形图 | 性别分布环形图.xlsx |
| 柱状图 | 柱状图 | 价格分布柱状图.xlsx |
| 漏斗图 | 漏斗图 | 转化漏斗图.xlsx |
| 气泡图 | 气泡图 | 价格评分气泡图.xlsx |
| 雷达图 | 雷达图 | 综合评价雷达图.xlsx |
| 矩形树图 | 矩形树图 | 品类分布矩形树图.xlsx |
| 词云 | 词云 | 热搜词云.xlsx |

未包含关键词的文件默认生成柱状图。

## 使用方法

### 启动方式

#### 方式一：打包版（推荐新手使用）

1. 双击 `启动.bat` 或直接运行 `和金顺数据平台.exe`
2. 在菜单中选择要启动的服务：

```
╔══════════════════════════════════════════════════════════════╗
║              和 金 顺 数 据 平 台                            ║
║         Hejinshun Data Platform v2.0                        ║
╚══════════════════════════════════════════════════════════════╝

┌──────────────────────────────────────────────────────────────┐
│  请选择要启动的服务:                                          │
├──────────────────────────────────────────────────────────────┤
│   [1] 数据大屏       - 电商数据可视化监控大屏 (port 5001)     │
│   [2] 数据工厂       - 结论驱动数据生成器 (port 5007)         │
│   [3] 代码生成器     - ECharts代码生成器 (port 5003)          │
│   [4] 全部启动       - 同时启动所有服务                       │
│   [0] 退出程序                                               │
└──────────────────────────────────────────────────────────────┘
```

3. 浏览器会自动打开对应页面；如未自动打开，手动访问以下地址：

| 服务 | 地址 |
|------|------|
| 数据大屏 | http://127.0.0.1:5001 |
| 数据工厂 | http://127.0.0.1:5007 |
| 代码生成器 | http://127.0.0.1:5003 |

#### 方式二：命令行参数启动

```bash
# 启动所有服务
和金顺数据平台.exe --all

# 只启动数据大屏
和金顺数据平台.exe --dashboard

# 只启动数据工厂
和金顺数据平台.exe --factory

# 只启动代码生成器
和金顺数据平台.exe --generator
```

#### 方式三：源码运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动数据大屏 (默认，端口 5001)
python main.py

# 启动数据工厂 (端口 5007)
python main.py --factory

# 启动代码生成器 (端口 5003)
python main.py --generator

# 指定端口
python main.py -p 8080

# 不自动打开浏览器
python main.py --no-browser
```

### 各功能操作步骤

#### 数据接入操作步骤

1. 启动数据大屏（方式一选 `[1]`，或 `python main.py`）
2. 在页面中切换到"数据接入"标签
3. 选择输入方式：
   - **关键词搜索**：输入关键词（如"漆器"），选择平台（京东/什么值得买）
   - **URL采集**：直接粘贴搜索页URL
4. 设置采集数量（建议5-50条，避免触发反爬虫）
5. 点击"开始采集"
6. Chrome浏览器会自动打开，等待采集完成
7. 采集完成后切换回"数据看板"标签查看数据

**注意事项：**
- 采集过程中不要关闭浏览器窗口
- 采集过程中不要手动操作浏览器
- 首次使用京东可能需要登录，请在弹出的浏览器窗口中完成登录

#### 数据工厂操作步骤

1. 启动数据工厂（方式一选 `[2]`，或 `python main.py --factory`）
2. 选择输入模式：
   - **结构化输入**：设置品类、价格、年龄、性别等参数
   - **自然语言输入**：输入结论语句（如"手机品类价格呈上涨趋势"）
3. 设置生成数量和输出格式
4. 点击"生成数据"，等待生成完成
5. 点击"导出到和金顺平台"将数据合并到主数据文件
6. 或点击"下载"保存文件到本地

#### 代码生成器操作步骤

1. 启动代码生成器（方式一选 `[3]`，或 `python main.py --generator`）
2. 上传一个或多个Excel文件（文件名包含图表类型关键词，如"饼图"）
3. 系统自动解析数据并生成ECharts配置
4. 复制生成的代码用于自己的项目

```
      │                                       │
      └──────────► data-factory ◄─────────────┘
                   数据制造工厂
```

### 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 后端 | Python 3.x + Flask | Web服务框架 |
| 前端 | HTML + CSS + JavaScript | 页面展示 |
| 图表 | ECharts 5.x | 数据可视化 |
| 爬虫 | DrissionPage | 浏览器自动化（Selenium备用） |
| 存储 | Excel (openpyxl) | 数据持久化，无数据库 |
| 数据生成 | NumPy | 统计数据生成 |

### 数据流向

```
crawler.scrape() → storage.clean_records() → storage.save_cleaned()
                                                       |
web.dashboard ← storage.read_cleaned() ←──────────────┘
                                                       ^
data-factory.generate() → export → ───────────────────┘
```

## 常见问题

### Q: 启动后浏览器没有自动打开？

手动在浏览器地址栏输入对应地址：
- 数据大屏：http://127.0.0.1:5001
- 数据工厂：http://127.0.0.1:5007
- 代码生成器：http://127.0.0.1:5003

### Q: 端口被占用怎么办？

**方法一**：关闭占用端口的程序。在命令行中查找并结束占用进程：

```bash
# 查找占用端口的进程
netstat -ano | findstr :5001

# 结束对应进程（替换PID）
taskkill /F /PID <进程ID>
```

**方法二**：使用 `-p` 参数指定其他端口：

```bash
python main.py -p 8080
```

### Q: 数据采集失败？

按以下顺序排查：

1. 检查网络连接是否正常
2. 确认URL格式正确（京东搜索页或什么值得买搜索页）
3. 减少采集数量（建议每次不超过20条）
4. 等待几分钟后重试（可能被反爬虫限制）
5. 首次使用京东需要登录：
   - Chrome浏览器会自动弹出
   - 在浏览器中登录京东账号
   - 登录成功后系统自动继续采集

### Q: 反爬虫被拦截？

- 等待10-15分钟后重试
- 更换网络环境（如使用手机热点）
- 先手动登录京东再采集
- 减少采集数量（建议每次不超过10条）

### Q: 采集过程中浏览器崩溃？

系统会自动重启浏览器并重试（最多2次）。如仍失败：
- 关闭所有Chrome窗口后重新启动采集
- 检查Chrome浏览器版本是否过旧

### Q: 数据工厂生成卡住？

点击"重置"按钮清除状态，然后重新操作。如问题持续，刷新网页重试。

### Q: 如何查看已有数据？

所有采集和导出的数据存储在 `data/cleaned/products.xlsx`，可直接用Excel打开查看。

### Q: 打包版exe启动闪退？

- 确认 `启动.bat` 和 `和金顺数据平台.exe` 在同一目录
- 右键 `启动.bat`，选择"以管理员身份运行"
- 检查杀毒软件是否拦截
